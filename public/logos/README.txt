Déposez ici les logos des certifications (noms exacts, en minuscules) :

  rge.png             → RGE - Reconnu Garant de l'Environnement
  qualipv.png         → QualiPV
  ademe.png           → ADEME

Vous pouvez utiliser .svg au lieu de .png ; dans ce cas, modifiez les chemins dans components/certifications-marquee.tsx (remplacez .png par .svg).

Dimensions recommandées (PNG) : hauteur 96–120 px, largeur au ratio. SVG : sans contrainte.

Sans ces fichiers, le bandeau affiche le nom du label en texte à la place du logo.
